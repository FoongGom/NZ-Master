# NZ-Master - NoiseZero DSP Simulation

## 하이브리드 층간소음 제어 시뮬레이션

본 코드는 층간소음을 유형별로 분류하고, 각 소음 특성에 맞는 DSP 제어 방식을 적용하여 저감 효과를 확인하는 시뮬레이션 코드입니다.

## 코드 파일

- `hybrid_floor_noise_control_simulation.py`

## 프로젝트 목적

층간소음을 완전히 제거하는 것이 아니라, 소음 유형에 따라 저감 가능성이 다르다는 점을 확인하고 각 유형에 맞는 제어 방식을 적용하는 것을 목표로 합니다.

## 소음 유형

본 시뮬레이션에서는 다음 5가지 층간소음을 가상으로 생성합니다.

- 아기 뛰는 소리
- 성인 발망치
- 세탁기 진동
- 의자 끄는 소리
- 물건 낙하 충격음

## 사용한 제어 방식

### 1. Fixed Gain/Delay

고정된 gain과 delay를 이용하여 반대 위상 제어 신호를 생성하는 방식입니다.  
구조가 단순하고 충격성 소음에서 비교적 안정적으로 동작할 수 있습니다.

### 2. FxNLMS Adaptive

스피커, 앰프, 판 전달 경로를 고려한 적응형 제어 방식입니다.  
세탁기 진동이나 의자 끄는 소리처럼 일정 시간 지속되는 소음에 효과적입니다.

### 3. Ringdown Impact Control

물건 낙하나 발망치처럼 순간적으로 발생하는 충격음에서 첫 충격 이후 남는 저주파 잔향을 줄이기 위한 방식입니다.

## 하이브리드 제어 선택 기준

소음 유형에 따라 다음과 같이 제어 방식을 선택합니다.

- 반복 진동: FxNLMS 적응형 제어
- 연속 마찰음: FxNLMS 적응형 제어
- 충격성 소음: Fixed Gain/Delay와 Ringdown 중 RMS 감소량이 더 좋은 방식 선택


## 한계점 및 개선 방향

시뮬레이션 결과, 세탁기 진동과 의자 끄는 소리처럼 지속 시간이 있는 소음은 비교적 잘 저감되었습니다.
하지만 아기 뛰는 소리, 성인 발망치, 물건 낙하처럼 짧고 강한 순간 충격성 소음은 저감 효과가 제한적이었습니다.

이는 다음과 같은 이유 때문입니다.

충격이 매우 짧은 시간에 발생함
센서가 감지하고 제어 신호를 출력하기 전에 첫 충격이 지나갈 수 있음
라즈베리파이 연산 지연, 앰프 지연, 스피커 응답 지연이 존재함
일반 스피커는 바닥판 진동을 직접 제어하기에 출력이 부족할 수 있음

따라서 실제 하드웨어 구현에서는 다음과 같은 개선이 필요합니다.

방진 패드 또는 고무 매트 사용
일반 스피커 대신 진동 액추에이터, Audio Exciter, Surface Transducer 사용
기준 센서와 오차 센서를 분리한 다중 센서 구조 적용
실제 실험 환경에 맞는 gain, delay, filter order, 학습률 튜닝
